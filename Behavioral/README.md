# Behavioral — STAR-истории для поведенческой части

Pet-проект №15 из чек-листа (не код). Закрывает раздел 17 из «200 ответов».
Здесь каркасы — наполни своими историями и **проговори вслух**: оценивают
гладкость подачи, а не идею. Одна история — 2–3 минуты речи.

## Формат STAR

- **S**ituation — контекст в 2 предложениях (проект, роль, ставки);
- **T**ask — что требовалось именно от тебя;
- **A**ction — что сделал ТЫ (не «мы»!), 60% времени истории;
- **R**esult — измеримый итог + чему научился.

Антипаттерны: «мы» вместо «я» · история без цифр в результате · виноваты все вокруг ·
идеальный герой без ошибок (не верят).

## 6 обязательных заготовок

### 1. Сложная техническая задача
Каркас: система деградировала / нетривиальный баг → как локализовал (метрики? трейс?
профайлер?) → гипотезы и проверка → решение → цифры до/после.
*Подсказка из твоего домена: инцидент с платежами/взысканием, N+1 или пул соединений под нагрузкой.*

### 2. Конфликт в команде
Каркас: разногласие по архитектуре/приоритетам → как перевёл спор из мнений в факты
(прототип, замер, ADR) → компромисс → отношения сохранены.
Ловушка вопроса: проверяют, умеешь ли не выигрывать спор, а решать проблему.

### 3. Своя ошибка / фейл
Каркас: конкретная ошибка (задеплоил, уронил, не проверил) → как обнаружил → что сделал
СРАЗУ (честно сказал, откатил) → системный фикс, чтобы не повторилось (тест, алерт, процесс).
Здесь нельзя юлить: история без реального фейла = красный флаг.

### 4. Менторинг / рост коллеги
Каркас: джун/новичок буксовал → как диагностировал, чего не хватает → формат помощи
(парное, код-ревью с объяснениями, план) → коллега стал самостоятелен (чем измерил).

### 5. Сорванный дедлайн / работа под давлением
Каркас: стало ясно, что не успеваем → как быстро поднял флаг (не молчал до конца!) →
что порезал (скоуп, не качество) → передоговорился со стейкхолдерами → выводы про оценки.

### 6. Технический долг
Каркас: долг мешал бизнесу (медленные релизы, инциденты) → как обосновал бизнесу
на языке денег/рисков → инкрементальный план (не «перепишем всё») → эффект в цифрах.
